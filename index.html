О проекте Спецификации Разработка Об авторе

Whiteplay Pocket OS

Официальная проектная документация

1. Простое описание проекта

Настоящий документ регламентирует процесс создания и разработки портативной вычислительной системы Whiteplay Pocket, а также специализированного программного обеспечения Whiteplay OS. Проект направлен на реализацию автономного устройства с кастомизируемым интерфейсом для запуска программного обеспечения и изучения архитектуры встраиваемых систем.

Whiteplay Pocket — это проект по созданию полноценной портативной игровой консоли и специализированной операционной системы для неё. Главная идея автора заключается в том, чтобы собрать с нуля полностью рабочее переносимое игровое устройство (по типу Nintendo Switch или PlayStation Vita), используя доступные электронные комплектующие.

Аппаратная база

Устройство проектируется как независимая игровая платформа. В качестве «сердца» консоли используется компактный вычислительный модуль Raspberry Pi Zero 2 W, за вывод изображения отвечает качественный 5-дюймовый IPS-экран, а элементы управления (кнопки и аналоговые стики) подключаются напрямую к плате. Все компоненты будут размещены в уникальном корпусе, разработанном специально для этого проекта.

Собственная операционная система

Главная особенность Whiteplay Pocket — это отказ от тяжелых готовых программ. Автор создает собственную операционную систему Whiteplay OS. Она базируется на специально облегченной версии Linux и пишется на языке C++ с графической библиотекой Raylib. Благодаря этому система будет запускаться за несколько секунд, работать плавно и тратить минимум заряда батареи.

Игры и цифровой магазин

Для консоли силами команды будут создаваться уникальные проекты — от простых пиксельных аркад до трехмерных гонок. Игры будут устанавливаться через собственный независимый магазин приложений, который связывает приставку с домашним сервером автора по беспроводной сети.

Образовательная ценность

Этот проект объединяет в себе конструирование железа, программирование и создание графики, позволяя участникам команды с нуля понять, как устроены современные гаджеты, и создать свое собственное уникальное устройство.

Перейдите по следующей ссылке, для получения информации о ролях в разработке: kivasily.github.io/whiteplay/simple-documentation.html

2. Технические спецификации оборудования

Аппаратная часть проектируется с учетом интеграции следующих компонентов:

  • Вычислительный модуль: Интегрированное решение на базе архитектуры ARM (Raspberry Pi Zero 2 W).
  • Дисплей: Цветная IPS-матрица (5 дюймов) с интерфейсом передачи данных HDMI.
  • Подсистема ввода: Аналоговые манипуляторы, тактовые кнопки направлений и действий, интегрированные через АЦП-конвертеры.
  • Модули расширения: Встроенный считыватель MicroSD для пользовательских данных и разъем 3.5 мм для вывода аудиосигнала.

3. Программная архитектура

Разработка операционной системы будет осуществляться в соответствии со следующими техническими стандартами:

  • Базовое ядро: Оптимизированная минималистичная сборка ядра ОС Linux для обеспечения высокой скорости инициализации аппаратных компонентов.
  • Графическая среда: Проектирование визуального интерфейса с использованием языка C++ и графической библиотеки Raylib.
  • Сетевая инфраструктура: Реализация клиент-серверного взаимодействия для цифрового магазина (HTML, CSS, Node.js) с поддержкой беспроводной передачи данных.
